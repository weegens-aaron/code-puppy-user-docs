Getting Started

Welcome to Code Puppy! This section will get you up and running quickly.

Quick Links

  • Installation — Install Code Puppy on your platform
  • Configuration — Set up your AI models and preferences
  • Quick Start — Your first coding task in 5 minutes

What You'll Need

Before you begin, make sure you have:

  1. Python 3.11+ installed
  2. uv package manager (recommended) or pip
  3. An AI provider API key (OpenAI, Anthropic, Google, Cerebras, or local model)

First Steps

  1. Install Code Puppy: uvx code-puppy -i
  2. Add a model: /add_model (interactive browser for 65+ providers)
  3. Start coding: Just type what you want to build!
