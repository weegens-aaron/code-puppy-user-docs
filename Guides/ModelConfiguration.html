How to Configure Models

What You'll Learn

By the end of this guide, you'll be able to configure model-specific settings like temperature, seed, and reasoning effort to fine-tune how Code Puppy interacts with each AI model.

Prerequisites

  • Code Puppy installed
  • At least one model added to your configuration (see AddModels guide)

Quick Version

# Open the interactive model settings menu
/model_settings

Navigate with arrow keys, press Enter to edit a setting, and Esc to save and exit.

Detailed Steps

1. Open the Model Settings Menu

Run the /model_settings command:

/model_settings

You'll see a split-screen interface:

  • Left panel: List of all available models (paginated if you have many)
  • Right panel: Details about the selected model
TIP

Models with custom settings show a ⚙ icon. The active model shows "(active)".

2. Select a Model to Configure

Use the ↑/↓ arrow keys to navigate through the model list. If you have more than 15 models, use PgUp/PgDn to change pages.

Press Enter on a model to view and edit its settings.

3. Configure Model Settings

You'll now see the settings available for that model. Navigate with ↑/↓ and press Enter to edit a setting.

Numeric Settings

For settings like Temperature, Seed, or Top-P:

  1. Press Enter to enter edit mode
  2. Use ←/→ arrow keys to adjust the value
  3. Press Enter to save, or Esc to cancel

Example: Setting temperature to 0.7:

  • Navigate to "Temperature"
  • Press Enter
  • Adjust with ←/→ until you see 0.70
  • Press Enter to save

Choice Settings

For settings like Reasoning Effort or Verbosity:

  1. Press Enter to enter edit mode
  2. Use ←/→ arrow keys to cycle through options
  3. Press Enter to save

Available options:

  • Reasoning Effort: minimal, low, medium, high, xhigh
  • Verbosity: low, medium, high, max
  • Extended Thinking: enabled, adaptive, off

Boolean Settings

For settings like Clear Thinking or Thinking Enabled:

  1. Press Enter to toggle between Enabled/Disabled
  2. Press Enter to save, or Esc to cancel

4. Reset a Setting to Default

While viewing a setting:

  • Press d to reset it to the model's default value

This removes the custom setting and reverts to the model's built-in default.

5. Exit and Save

  • Press Esc to go back to the model list
  • Press Esc again to exit the menu

All changes are saved automatically as you make them.

Understanding Model Settings

Available Settings

Setting Type Description Typical Values
Temperature Numeric (0.0-1.0) Controls randomness. Lower = more deterministic, higher = more creative 0.2 (precise) to 0.8 (creative)
Seed Numeric (0-999999) Random seed for reproducible outputs. Set same value for consistent results Any integer
Top-P Numeric (0.0-1.0) Controls token diversity. 0.0 = least random, 1.0 = most random 0.9 (common default)
Reasoning Effort Choice How much effort GPT-5 models spend on reasoning medium (balanced)
Verbosity Choice Controls response length medium (balanced)
Extended Thinking Choice Enable thinking mode with budget_tokens enabled (classic) or adaptive (model decides)
Thinking Budget Numeric (1024-131072) Max tokens for extended thinking (when enabled) 10000 (default)
Interleaved Thinking Boolean Enable thinking between tool calls (Claude 4 only) false (default)
Clear Thinking Boolean Strip thinking blocks from responses false (show thinking)
Thinking Enabled Boolean Enable thinking mode for Gemini 3 Pro true (default)
Thinking Level Choice Depth of thinking for Gemini 3 Pro low (faster) or high (thorough)
Effort Choice Response effort for Opus 4-6 high (default)
NOTE

Not all models support all settings. The menu only shows settings that are valid for the selected model.

Global vs. Per-Model Settings

Some settings are global (apply to all models of a type):

  • Reasoning Effort: Applies to all GPT-5 models
  • Verbosity: Applies to all OpenAI models

When you configure these for a specific model, they affect all models of that type.

Examples

Example 1: Make a Model More Creative

Configure claude-3-5-sonnet for creative brainstorming:

  1. /model_settings
  2. Select claude-3-5-sonnet
  3. Set Temperature to 0.8
  4. Set Top-P to 0.95

Result: The model will produce more varied and creative responses.

Example 2: Make a Model More Precise

Configure gpt-5 for code review:

  1. /model_settings
  2. Select gpt-5
  3. Set Temperature to 0.2
  4. Set Reasoning Effort to high

Result: The model will be more deterministic and thorough in its analysis.

Example 3: Reproduce Results

Set a fixed seed for reproducible outputs:

  1. /model_settings
  2. Select your model
  3. Set Seed to 42

Result: Running the same prompt will produce consistent results.

Example 4: Enable Extended Thinking

Configure a Claude model for deep reasoning:

  1. /model_settings
  2. Select your Claude model
  3. Set Extended Thinking to enabled
  4. Set Thinking Budget to 20000

Result: The model will show its reasoning process with a 20k token budget.

Options & Variations

Command-Line Configuration

You can also set some model settings via /set commands:

# Set global temperature
/set temperature 0.7

# Set reasoning effort (global for GPT-5)
/set openai_reasoning_effort high

# Set verbosity (global for OpenAI)
/set openai_verbosity low
NOTE

The /set command only supports a subset of settings. For full control, use /model_settings.

Programmatic Configuration

You can configure models in your plugins using the config API:

from code_puppy.config import set_model_setting

# Set temperature for a specific model
set_model_setting("gpt-5", "temperature", 0.7)

# Clear a setting (revert to default)
set_model_setting("gpt-5", "temperature", None)

Troubleshooting

Problem Cause Solution
Setting not available for my model Model doesn't support that setting Check the model's supported_settings in models.json
Changes not taking effect Model was already loaded Use /reload or restart Code Puppy
Can't find my model Model not added to models.json Use /add_model to add it first
Settings reset after restart Settings not saved Changes are saved automatically; check you're editing the right model
