How to Switch Agents

What You'll Learn

By the end of this guide, you'll be able to switch between different specialized agents and configure which AI model each agent uses.

Prerequisites

  • Code Puppy installed and running
  • Familiarity with basic slash commands (see Commands Guide)

Quick Version

/agent

Press Enter to open the interactive agent picker, then navigate and select an agent.

Detailed Steps

1. Open the Agent Picker

Type /agent (or the shortcut /a) and press Enter:

/agent

This opens an interactive terminal UI with two panels:

  • Left panel: List of all available agents
  • Right panel: Details about the currently selected agent

2. Navigate the Agent List

Use these keys to browse:

Key Action
↑ ↓ Move selection up/down
← → Change page (10 agents per page)
Enter Switch to selected agent
Ctrl+C Cancel without switching

The current active agent is marked with ← current.

3. View Agent Details

As you navigate, the right panel shows:

  • Name: Internal identifier (e.g., code_puppy)
  • Display Name: Human-readable name (e.g., Code Puppy)
  • Pinned Model: The AI model assigned to this agent (or "default")
  • Description: What the agent specializes in

4. Pin a Model to an Agent

You can assign a specific AI model to an agent so it always uses that model:

  1. Navigate to the agent you want to configure
  2. Press P to open the model picker
  3. Select a model from the list
  4. The model is saved and the agent is reloaded if it's currently active

To unpin a model, press P again and select (unpin).

TIP

Pinned models persist across sessions. The agent will always use your pinned model until you unpin it.

5. Clone an Agent

Create a custom copy of an agent with its own settings:

  1. Navigate to the agent you want to clone
  2. Press C to create a clone
  3. The clone is created with a unique name (e.g., code_puppy_copy_1)
  4. You can now pin a different model to the clone

6. Delete a Clone

Remove a cloned agent you no longer need:

  1. Navigate to the clone you want to delete
  2. Press D to delete it
  3. Confirm the deletion
WARNING

- Only cloned agents can be deleted - You cannot delete the currently active agent - You cannot delete built-in agents

Example: Switching to a Specialized Agent

Let's say you're working on a C++ project and want to use the C++ reviewer:

/agent

Navigate to cpp_reviewer using the arrow keys, then press Enter:

✓ Switched to agent: C++ Reviewer
C++ code review specialist with focus on memory safety and performance
Auto-save session rotated to: abc123-def456

Now all your interactions will be handled by the C++ Reviewer agent.

Available Agents

Code Puppy comes with several built-in agents:

Agent Purpose
code_puppy General-purpose coding assistant
code_reviewer Multi-language code review
c_reviewer C language specialist
cpp_reviewer C++ language specialist
golang_reviewer Go language specialist
javascript_reviewer JavaScript/Node.js specialist
typescript_reviewer TypeScript specialist
python_reviewer Python language specialist
python_programmer Python code generation
security_auditor Security-focused code analysis
qa_expert Quality assurance and testing
terminal_qa Command-line and shell expert
planning Task planning and breakdown
scheduler Task scheduling and coordination

You can also create custom agents using JSON configuration files (see Create Agents Guide).

Options & Variations

Direct Agent Switching

You can switch to an agent directly by name:

/agent cpp_reviewer

This skips the picker and immediately switches to the C++ Reviewer.

Model Distribution

For advanced multi-model setups, see Round-Robin Models Guide.

Troubleshooting

Problem Cause Solution
"No agents found" Agent registry issue Restart Code Puppy
Agent picker doesn't open Terminal compatibility Try /agent <name> directly
"Cannot delete the active agent" Trying to delete current agent Switch to a different agent first
Model picker shows no models No models configured Add models via /add-model
