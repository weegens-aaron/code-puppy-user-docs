Code Puppy

What Is Code Puppy?

Code Puppy is an AI-powered code generation agent that helps you write, review, and understand code through natural language conversations. Think of it as having a team of specialized AI developers at your command—each with their own personality and expertise.

Unlike traditional IDEs or single-model AI assistants, Code Puppy gives you 1024+ different "puppies" (specialized agents) that can work simultaneously, distributed across multiple AI models to overcome rate limits and maximize productivity. It runs entirely in your terminal with a sassy, opinionated personality that keeps code quality high.

Code Puppy is designed for developers who want:

  • Privacy-first AI assistance - Your code never leaves your control unless you choose an external provider
  • Multi-model distribution - Spread load across dozens of AI providers
  • Specialized agents - Switch between general-purpose coders, code reviewers, security auditors, and more
  • Extensible architecture - Add custom skills, hooks, and plugins without modifying core code

Key Features

Feature Description
Multi-Agent System Switch between 20+ specialized agents (code reviewer, security auditor, Python programmer, etc.)
65+ AI Providers Access 1000+ models from OpenAI, Anthropic, Google, Cerebras, and 60+ more providers
Agent Skills Install reusable capability packs (Docker, Kubernetes, security audits, etc.)
MCP Integration Connect to Model Context Protocol servers for extended tooling
Custom Hooks Intercept and control every tool call with pre/post hooks
Round-Robin Models Distribute requests across multiple model instances to bypass rate limits
Privacy-First Zero telemetry, no prompt logging, no third-party data sharing
100% Open Source Fully auditable MIT-licensed code

Requirements

NOTE

Code Puppy is a CLI tool that runs locally on your machine. You need Python 3.11+ and at least one AI provider API key.

Requirement Details
OS macOS, Linux, or Windows (with PowerShell)
Python 3.11 or higher
Package Manager [uv](https://github.com/astral-sh/uv) (recommended) or pip
AI Provider At least one: OpenAI, Anthropic, Google, Cerebras, or local model server

Getting Started

Quick Install

# Install and run in one command (recommended)
uvx code-puppy -i

# Or install globally as a tool
uv pip install code-puppy
code-puppy -i

First Run

When you first run Code Puppy, you'll see:

  1. A message of the day (MOTD) with latest updates
  2. The interactive terminal interface
  3. Available commands via /help

Configure an AI Model

You need at least one AI provider configured. The easiest way is:

# Add models from models.dev (65+ providers)
/add_model

Or set environment variables for built-in providers:

# OpenAI
export OPENAI_API_KEY=sk-...

# Anthropic (Claude)
export ANTHROPIC_API_KEY=sk-ant-...

# Google (Gemini)
export GEMINI_API_KEY=AIza...

# Cerebras
export CEREBRAS_API_KEY=csk-...

# Synthetic (recommended for best value)
export SYN_API_KEY=...

Accessing Code Puppy

Code Puppy runs in your terminal:

# Interactive mode (default)
uvx code-puppy -i

# Run a single prompt and exit
uvx code-puppy -p "Write a Python function to calculate fibonacci"

# Use a specific agent
uvx code-puppy -i --agent code-puppy

# Use a specific model
uvx code-puppy -i --model gpt-5.2
