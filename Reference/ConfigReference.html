Configuration Reference

Overview

This reference covers all configuration options in Code Puppy's puppy.cfg file. Configuration controls agent behavior, model settings, and system features.

Configuration File Location

Location Path
Default ~/.code_puppy/puppy.cfg
XDG Config $XDG_CONFIG_HOME/code_puppy/puppy.cfg

Format: INI-style configuration file

Core Settings

puppy_name

Description: Name for your Code Puppy instance.

Required: Yes

Example:

puppy_name = Code Puppy

owner_name

Description: Your name (so Code Puppy knows its owner).

Required: Yes

Example:

owner_name = John Doe

model

Description: Default model to use for agent responses.

Default: First model in models.json

Example:

model = gpt-5

Related:

  • Reference: Models

temperature

Description: Global model temperature (creativity level).

Range: 0.0 to 2.0

Default: Model-specific default

Example:

temperature = 0.7

Per-Model Alternative:

model_settings_gpt_5_temperature = 0.7
model_settings_claude_4_5_sonnet_temperature = 0.5

Related:

  • Reference: Model Settings

yolo_mode

Description: Enable "YOLO mode" for aggressive code changes.

Values: true, false, 1, 0, yes, no, on, off

Default: true

Example:

yolo_mode = true

Behavior:

  • When true: Code Puppy makes changes without additional confirmation
  • When false: Requires confirmation for potentially destructive operations

enable_dbos

Description: Enable DBOS (Durable Object State) for state persistence.

Values: true, false, 1, 0, yes, no, on, off

Default: true

Example:

enable_dbos = true

enable_pack_agents

Description: Enable pack agents (specialized sub-agents).

Values: true, false, 1, 0, yes, no, on, off

Default: false

Example:

enable_pack_agents = true

Pack Agents:

  • pack-leader: Coordinates pack operations
  • bloodhound: Searches and discovers
  • husky: Handles heavy lifting
  • shepherd: Manages workflows
  • terrier: Small, focused tasks
  • watchdog: Monitors and alerts
  • retriever: Fetches information

enable_universal_constructor

Description: Enable the Universal Constructor tool for dynamic tool creation.

Values: true, false, 1, 0, yes, no, on, off

Default: true

Example:

enable_universal_constructor = true

Behavior:

  • When true: Agents can create custom tools at runtime
  • When false: Universal Constructor tool is not available

enable_streaming

Description: Enable streaming (SSE) for model responses.

Values: true, false, 1, 0, yes, no, on, off

Default: true

Example:

enable_streaming = true

Sub-Agent Settings

subagent_verbose

Description: Enable verbose output for sub-agents.

Values: true, false, 1, 0, yes, no, on, off

Default: false

Example:

subagent_verbose = false

Behavior:

  • When false: Sub-agents produce quiet, sparse output
  • When true: Sub-agents produce full verbose output (useful for debugging)

OpenAI-Specific Settings

openai_reasoning_effort

Description: Reasoning effort level for OpenAI models.

Values: minimal, low, medium, high, xhigh

Default: medium

Example:

openai_reasoning_effort = high

openai_verbosity

Description: Verbosity level for OpenAI model responses.

Values: low, medium, high

Default: medium

Example:

openai_verbosity = low

Behavior:

  • low: More concise responses
  • medium: Balanced responses
  • high: More verbose responses

Session & History Settings

auto_save_session

Description: Automatically save session state.

Values: true, false, 1, 0, yes, no, on, off

Default: true

Example:

auto_save_session = true

max_saved_sessions

Description: Maximum number of sessions to keep saved.

Default: Not set (unlimited)

Example:

max_saved_sessions = 10

cancel_agent_key

Description: Key sequence to cancel an agent.

Default: Not set

Example:

cancel_agent_key = Ctrl+C

Token & Context Settings

protected_token_count

Description: Number of tokens to protect from compaction.

Default: Not set

Example:

protected_token_count = 10000

compaction_threshold

Description: Threshold for triggering message compaction.

Default: Not set

Example:

compaction_threshold = 100000

compaction_strategy

Description: Strategy for message compaction.

Default: Not set

Example:

compaction_strategy = summarize

message_limit

Description: Maximum number of messages to keep in history.

Default: Not set

Example:

message_limit = 100

MCP Settings

disable_mcp

Description: Disable all MCP servers.

Values: true, false, 1, 0, yes, no, on, off

Default: false

Example:

disable_mcp = false

Behavior:

  • When true: Code Puppy skips loading MCP servers entirely
  • When false: MCP servers are loaded normally

Output Settings

grep_output_verbose

Description: Enable verbose output for grep operations.

Values: true, false, 1, 0, yes, no, on, off

Default: false

Example:

grep_output_verbose = true

Behavior:

  • When false: Shows only file names with match counts
  • When true: Shows full output with line numbers and content

diff_context_lines

Description: Number of context lines to show in diffs.

Default: Not set

Example:

diff_context_lines = 3

Frontend Settings

frontend_emitter_enabled

Description: Enable frontend event emitter.

Values: true, false, 1, 0, yes, no, on, off

Default: Not set

Example:

frontend_emitter_enabled = true

frontend_emitter_max_recent_events

Description: Maximum number of recent events to keep.

Default: Not set

Example:

frontend_emitter_max_recent_events = 100

frontend_emitter_queue_size

Description: Size of the event queue.

Default: Not set

Example:

frontend_emitter_queue_size = 1000

Agent Settings

default_agent

Description: Default agent to use.

Default: Not set

Example:

default_agent = code-puppy

Model-Specific Settings

Per-Model Settings

You can configure settings for specific models using the pattern:

model_settings_<model_name>_<setting> = <value>

Example:

# Temperature for gpt-5
model_settings_gpt_5_temperature = 0.7

# Temperature for claude-4-5-sonnet
model_settings_claude_4_5_sonnet_temperature = 0.5

# Top-p for gpt-5
model_settings_gpt_5_top_p = 0.9

# Seed for reproducible results
model_settings_gpt-5_seed = 42

Supported Settings:

Setting Type Description
temperature float Creativity (0.0-2.0)
top_p float Nucleus sampling (0.0-1.0)
seed int Reproducible randomness
budget_tokens int Extended thinking budget
clear_thinking bool Enable clear thinking (GLM models)

Model Name Sanitization:

  • Dots (.) become underscores (_)
  • Hyphens (-) become underscores (_)
  • Forward slashes (/) become underscores (_)
  • Converted to lowercase

Examples:

Model Name Config Key Prefix
gpt-5 model_settings_gpt_5_
claude-4-5-sonnet model_settings_claude_4_5_sonnet_
gemini-2.5-flash model_settings_gemini_2_5_flash_

Banner Settings

banner_color_<name>

Description: Custom banner color.

Example:

banner_color_default = magenta
banner_color_error = red
banner_color_success = green

Resume Settings

resume_message_count

Description: Number of messages to show when resuming a session.

Default: Not set

Example:

resume_message_count = 10

Safety Settings

safety_permission_level

Description: Safety permission level for risky operations.

Values: none, low, medium, high, critical

Default: medium

Example:

safety_permission_level = high

Levels:

  • none: No safety checks
  • low: Minimal safety checks
  • medium: Standard safety checks
  • high: Strict safety checks
  • critical: Maximum safety checks

Allow Settings

allow_recursion

Description: Allow recursive agent calls.

Values: true, false, 1, 0, yes, no, on, off

Default: true

Example:

allow_recursion = true

HTTP Settings

http2

Description: Enable HTTP/2 support.

Values: true, false, 1, 0, yes, no, on, off

Default: Not set

Example:

http2 = true

Authentication

puppy_token

Description: Authentication token for Code Puppy services.

Example:

puppy_token = your_token_here

Example Configuration

[puppy]
# Core settings (required)
puppy_name = Code Puppy
owner_name = John Doe

# Model settings
model = gpt-5
temperature = 0.7

# Behavior settings
yolo_mode = true
enable_dbos = true
enable_pack_agents = false
enable_universal_constructor = true
enable_streaming = true

# Sub-agent settings
subagent_verbose = false

# OpenAI settings
openai_reasoning_effort = medium
openai_verbosity = medium

# Session settings
auto_save_session = true
max_saved_sessions = 10

# Token settings
protected_token_count = 10000
compaction_threshold = 100000

# MCP settings
disable_mcp = false

# Output settings
grep_output_verbose = false
diff_context_lines = 3

# Safety settings
safety_permission_level = medium

# Allow settings
allow_recursion = true

# Per-model settings
model_settings_gpt_5_temperature = 0.7
model_settings_claude_4_5_sonnet_temperature = 0.5

Managing Configuration

View Current Settings

# View all settings
cat ~/.code_puppy/puppy.cfg

Set a Setting

# Using the CLI
/code_puppy config set model gpt-5
/code_puppy config set temperature 0.7

Reset a Setting

# Remove a setting to reset to default
/code_puppy config reset model
