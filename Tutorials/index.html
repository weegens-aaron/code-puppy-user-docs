What's a Tutorial?

Tutorials are learning experiences that walk you through complete scenarios. Unlike guides (which help you do a specific task), tutorials teach you by doing.

Recommended Order

If you're new to Code Puppy:

  1. Start with Your First Code Task
  2. Then explore the Guides for specific features
  3. Try Multi-Model Setup when you're ready to scale
  4. Create your own Custom Agent when you need specialization
