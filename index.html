Code Puppy Documentation

Welcome!

Code Puppy is an AI-powered code generation agent that runs in your terminal. It helps you write, review, and understand code through natural language conversations—with a sassy personality and 1024+ specialized "puppies" (agents) at your command.

Quick Command Reference

# Start Code Puppy
uvx code-puppy -i

# Get help
/help

# Switch agents
/agent <name>

# Add a model from models.dev
/add_model

# Install skills
/skills

# Manage MCP servers
/mcp

# Exit
/exit
